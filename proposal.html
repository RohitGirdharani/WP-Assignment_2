Project Description


It is a web based application for hospitals in order to manage doctors and patients, through which various functionalities can be performed by the users. We intend to build this project using Bootstrap, PHP and MYSQL Database. This system consist of three modules:



  1. Admin Module
  2. Doctor Module
  3. Patient Module


Admin Module



  1. Admin can view the Patients, Doctors, Appointments and all new queries.
  2. Admin can add and update Doctors
  3. View user's data and delete the irrelevant user
  4. Admin can search any patient with the help of patient name and mobile number.
  5. Admin can check login and logout time of both doctors and patients.

Patient Module



  1. Patient can view their profile
  2. Patient can book their appointments
  3. Patient can book their appointments
  4. Patient can view their appointment history

Doctor Module



  1. Doctors can view their own appointments and history
  2. Doctors can view patient's appointment history
  3. Doctors can search patient by name or mobile number
  4. Doctors can manage patients (add or update)
